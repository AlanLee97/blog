# 作业提醒APP

作业题共享提醒 APP是一个作业提醒的Android APP。

功能：记录作业、作业共享、定时提醒完成作业、番茄时钟、短信验证、获取当前天气信息等功能。

后端采用ThinkPHP框架。



## 技术桟

| 名称     | 说明                    | 版本      |
| -------- | ----------------------- | --------- |
| QMUI     | UI库                    | 1.4.0     |
| banner   | 轮播图组件              | 1.4.10    |
| Okhttp3  | 网络请求框架            | 3.10.0    |
| Gson     | JSON序列化工具          | 2.8.5     |
| bmob-sdk | Bmob后端云SDK——短信验证 | 3.7.3-rc1 |
| ThinkPHP | 后端服务框架            | 5.0       |
