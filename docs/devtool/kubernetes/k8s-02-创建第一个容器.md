# 创建第一个容器

以安装nginx为例

## 创建容器

### 在master机器上操作

**创建容器**

```sh
kubectl run nginx --image=nginx --replicas=2 --port=80
```

--image=nginx 指定镜像为nginx

--replicas=2 创建实例的数量为2

--port=80 指定容器端口为80

```sh

```

--port=80 容器的端口为80

--type=LoadBalancer 类型为负载均衡

## 删除容器

### 在master节点上操作

**删除部署**

```sh
kubectl delete deployment nginx
```
