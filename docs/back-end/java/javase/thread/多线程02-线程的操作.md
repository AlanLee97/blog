# 线程的操作

## 线程命名

### 知识点

通过下面代码获取线程名字

```java
Thread.currentThread().getName()
```



### 实例

**代码**

```java
public class TestThread {
    public static void main(String[] args) {
        System.out.println("当前线程：" + Thread.currentThread().getName());

        //未自定义名字的线程
        new Thread(() -> {
            System.out.println("当前线程：" + Thread.currentThread().getName());
        }).start();

        //自定义名字的线程
        new Thread(() -> {
            System.out.println("当前线程：" + Thread.currentThread().getName());
        }, "线程A").start();

    }
}
```

**结果**

```java
当前线程：main
当前线程：Thread-0
当前线程：线程A
```





## 线程睡眠

###  知识点

通过下面代码使线程睡眠

```
Thread.sleep(1000);
```

该方法必须处理异常

### 实例

**代码**

```java
public class TestThread2 {
    public static void main(String[] args) {
        new Thread(() -> {
            for (int i = 0; i < 5; i++) {
                System.out.println(i);
                //线程休眠
                try {
                    Thread.sleep(1000);
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        }).start();
    }
}
```



**结果**

```java
0
1
2
3
4
```



## 线程中断

### 知识点

中断线程

```
public void interrupt()
```

判断线程是否被中断

```
public boolean isInterrupted()
```



### 实例

**代码**

```java
public class TestThread3 {
    public static void main(String[] args) throws InterruptedException {
        Thread thread = new Thread(() -> {
            for (int i = 0; i < 5; i++) {
                System.out.println(i);
                try {
                    Thread.sleep(1000);
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        });

        thread.start();
        if (!thread.isInterrupted()){
            thread.interrupt();
            System.out.println("线程被中断");
        }
    }
}
```



**结果**






## 线程强制运行

### 知识点

当一个线程满足某些条件之后，这个线程对象将可以一直独占资源，一直到该线程的程序执行结束。

使用`join()`方法



### 实例

**代码**

```java
//线程强制执行
public class TestThread4 {
    public static void main(String[] args) {
        Thread mainThread = Thread.currentThread();

        //子线程
        new Thread(() -> {
            for (int i = 0; i < 10; i++) {
                if (i == 5){
                    try {
                        System.out.println("主线程加入进来了，让主线程执行完所有代码，再执行子线程的代码！！");
                        mainThread.join();
                        System.out.println("主线程执行完了，子线程继续执行子线程里面的代码");
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }
                System.out.println("A线程：" + i);
                try {
                    Thread.sleep(500);
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        }).start();



        //主线程
        for (int i = 0; i < 10; i++) {
            System.out.println("主线程：" + i);
            try {
                Thread.sleep(500);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
    }
}

```

## 线程礼让

### 知识点

线程的礼让指的是先将资源让出去让别的线程先执行。

使用`yield()`方法

### 实例

**代码**

```java
//线程强制执行
public class TestThread5 {
    public static void main(String[] args) {

        //A线程
        new Thread(() -> {
            for (int i = 1; i <= 10; i++) {
                if (i % 2 == 0){
                    System.out.println("A线程将资源礼让出去，让其他线程先执行");
                    Thread.yield();
                    System.out.println("资源礼让完成，A线程拿回资源继续执行");
                }
                System.out.println("A线程：" + i);
                try {
                    Thread.sleep(500);
                } catch (InterruptedException e) {
                    e.printStackTrace();
                }
            }
        }).start();


        //主线程
        for (int i = 1; i <= 10; i++) {
            System.out.println("主线程：" + i);
            try {
                Thread.sleep(500);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }


    }
}
```

## 线程优先级

### 知识点

从理论上来讲，线程的优先级越高，越有可能先执行（越有可能先抢占到资源）。

优先级的处理方法：

- 设置优先级 public final void setPriority(int newPriority)
- 获取优先级 public final int getPriority()

线程的优先级有3个常量

- 最小优先级（默认值为 1）

```
public final static int MIN_PRIORITY
```

- 中等优先级（默认值为 5）

```
public final static int NORM_PRIORITY
```

- 最大优先级（默认值为 10）

```
public final static int MAX_PRIORITY
```

### 实例

**代码**

```java
//线程的优先级
public class TestThread6 {
    public static void main(String[] args) {

        //A线程：设置A线程的优先级为最小优先级
        Thread aThread = new Thread(() -> {
            for (int i = 0; i < 4; i++) {
                System.out.println("A线程：" + i);
            }
        }, "A线程");
        aThread.start();
        aThread.setPriority(Thread.MIN_PRIORITY);
        System.out.println("A线程的优先级：" + aThread.getPriority());

        //B线程：设置B线程的优先级为最大优先级
        Thread bThread = new Thread(() -> {
            for (int i = 0; i < 4; i++) {
                System.out.println("B线程：" + i);
            }
        }, "B线程");
        bThread.setPriority(Thread.MAX_PRIORITY);
        bThread.start();
        System.out.println("B线程的优先级：" + bThread.getPriority());

        //主线程
        Thread mainThread = new Thread();
        System.out.println("主线程的优先级：" + mainThread.getPriority());
        for (int i = 0; i < 4; i++) {
            System.out.println("主线程：" + i);
        }
    }
}

```
